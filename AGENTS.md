# Agnes AI 在线演示平台 - 需求拆解文档

## 产品概述

- **产品类型**: AI 能力演示平台（工具类应用）
- **场景类型**: <scene_type>prototype-app</scene_type>
- **目标用户**: 对 Agnes AI 模型能力感兴趣的开发者、产品经理、技术决策者，希望通过在线体验快速了解模型效果
- **核心价值**: 提供一站式 Agnes AI 三大核心模型（文本/图像/视频）的在线演示体验，降低试用门槛
- **界面语言**: 中文
- **主题偏好**: 深色（用户明确要求"现代科技感深色主题"）
- **导航模式**: 路径导航（多页面系统，功能模块独立）
- **导航布局**: Sidebar（内部工具/中后台风格，功能模块切换）

## 页面布局建议

- **布局模式**: 全局 Sidebar + 主内容区（左侧固定导航，右侧内容区）。工具类应用，功能模块独立，Sidebar 提供清晰的模块切换入口
- **视觉重心**: 结果承载区（对话历史/生成图片/生成视频）—— 用户核心目的是体验模型输出效果，结果展示是页面最重要的区域
- **结果承载区**:
  - **文本聊天页**: 对话消息列表（用户消息 + AI 流式回复），初始态为欢迎语 + 输入框引导
  - **图像生成页**: 图片预览卡片（含下载按钮），初始态为空状态提示"输入提示词开始生成"
  - **视频生成页**: 视频播放器（含下载按钮），初始态为空状态提示"输入描述文本开始生成视频"
- **源材料承载区**: 无（输入仅为文本 prompt，不需要持续参照原始材料）

## 数据来源声明

| 数据/操作 | 来源类型 | 实现要求 | mock 兜底 |
|---|---|---|---|
| API Token 配置 | local-persist | localStorage key=`__agnes_demo_token`，读取/保存用户输入的 API Key | 无（首次使用为空，引导用户输入） |
| 文本聊天（发送消息、流式接收回复） | real-api | 调用 `https://apihub.agnes-ai.com/v1/chat/completions`，model=`Agnes-2.0-Flash`，stream=true，使用 OpenAI 兼容格式 | 无（API 不可用时 toast 提示"请检查 API Token 和网络连接"） |
| 图像生成（提交 prompt、获取图片） | real-api | 调用 `https://apihub.agnes-ai.com/v1/images/generations`，model=`Agnes-Image-2.1-Flash`，使用 OpenAI 兼容格式 | 无（API 不可用时 toast 提示"图像生成失败，请检查 Token"） |
| 视频生成（提交文本、获取视频） | real-api | 调用 `https://apihub.agnes-ai.com/v1/video/generations`，model=`Agnes-Video-V2.0`，使用 OpenAI 兼容格式 | 无（API 不可用时 toast 提示"视频生成失败，请检查 Token"） |
| 对话历史（文本聊天） | local-persist | localStorage key=`__agnes_demo_chat_history`，保存当前会话的消息列表，页面刷新后恢复 | 无（首次使用为空数组） |
| 生成的图片/视频下载 | import-export | 图片：fetch → Blob → `URL.createObjectURL` + `<a>` click 触发下载；视频同理 | 无 |
| 首页模型能力介绍 | demo-mock | 静态内容，直接硬编码在 `HomePage.tsx` 中 | ✅ 本身就是 mock |

> **说明**：文本聊天、图像生成、视频生成均为真实 API 调用，依赖用户提供的 API Token。Token 未配置时，各功能页应引导用户先配置 Token（提供跳转设置面板的入口）。

## 功能列表

> **说明**：每个页面的功能点，供页面生成使用

### 全局组件：API Token 配置面板

- **组件目标**: 让用户输入并保存 Agnes AI API Key，所有 API 调用依赖此 Token
- **功能点**:
  - **Token 输入与保存**: 提供输入框（type=password 掩码显示），用户输入后点击保存，写入 localStorage。保存成功后 toast 提示"Token 已保存"
  - **可折叠设置面板**: 默认折叠，点击设置图标/按钮展开。面板内显示当前 Token 状态（已配置/未配置）、基础 URL（固定显示 `https://apihub.agnes-ai.com/v1`，不可编辑）、Token 输入框、保存/清除按钮
  - **Token 清除**: 提供"清除 Token"按钮，确认后删除 localStorage 中的 Token，toast 提示"Token 已清除"
  - **Token 状态指示**: 在全局 UI（如 Sidebar 底部）显示 Token 配置状态图标（绿色圆点=已配置，红色圆点=未配置），点击可打开设置面板

---

### 页面: 首页/模型介绍 (`/`)

- **页面目标**: 展示 Agnes AI 三大核心模型能力，引导用户进入各功能模块体验
- **功能点**:
  - **Hero 区域**: 展示平台标题"Agnes AI 在线演示平台"、副标题简介，视觉冲击力强的渐变背景
  - **三大能力卡片**: 三张玻璃拟态卡片，分别展示：
    - 📝 文本生成与推理 — Agnes-2.0-Flash 高效语言模型
    - 🎨 图像生成与编辑 — Agnes-Image-2.1-Flash 高性能图像模型
    - 🎬 音视频同步生成 — Agnes-Video-V2.0 电影级视频生成
  - **卡片交互**: 每张卡片包含"立即体验"按钮，点击跳转到对应功能页（`/chat`、`/image-gen`、`/video-gen`）
  - **Token 状态提示**: 若 Token 未配置，在首页顶部显示友好提示条"👋 请先配置 API Token 以开始体验"，点击可打开设置面板

---

### 页面: 文本聊天 (`/chat`)

- **页面目标**: 提供多轮对话界面，用户输入消息后流式展示 AI 回复
- **功能点**:
  - **对话消息列表**: 展示用户消息（右侧气泡）和 AI 回复（左侧气泡），支持滚动到底部。AI 回复支持 Markdown 渲染（代码块、列表等）
  - **流式输出**: AI 回复以打字机效果逐字显示，使用 SSE 流式解析 `data: [DONE]` 结束标记
  - **消息输入**: 底部固定输入框 + 发送按钮，支持 Enter 发送（Shift+Enter 换行）。发送后输入框清空，显示加载状态（发送按钮变为加载动画）
  - **对话历史持久化**: 页面刷新后恢复上次对话记录（从 localStorage 读取），提供"清空对话"按钮（确认后清除历史）
  - **Token 未配置引导**: 若 Token 未配置，输入框禁用，显示提示"请先配置 API Token"，点击可打开设置面板
  - **错误处理**: API 调用失败时，在对话列表中显示错误消息气泡（红色提示），不中断对话流程

---

### 页面: 图像生成 (`/image-gen`)

- **页面目标**: 用户输入提示词，生成并展示 AI 图片，支持预览和下载
- **功能点**:
  - **提示词输入**: 输入框 + 生成按钮，用户输入图片描述文本（支持中文/英文）
  - **生成状态**: 点击生成后按钮显示加载动画，提示"正在生成图片，请稍候..."
  - **图片结果展示**: 生成成功后展示图片预览卡片（玻璃拟态容器），图片自适应宽度，点击可放大预览（Lightbox 效果）
  - **图片下载**: 预览卡片上提供下载按钮，点击触发浏览器下载（文件名：`agnes-image-{timestamp}.png`）
  - **生成历史**: 当前会话中生成的图片以缩略图列表形式展示在结果区下方，支持切换查看
  - **Token 未配置引导**: 若 Token 未配置，输入框禁用，显示提示"请先配置 API Token"
  - **错误处理**: 生成失败时 toast 提示具体错误信息，输入框保持可用状态

---

### 页面: 视频生成 (`/video-gen`)

- **页面目标**: 用户输入文本描述，生成并展示 AI 视频，支持预览和下载
- **功能点**:
  - **文本描述输入**: 输入框 + 生成按钮，用户输入视频描述文本
  - **生成状态**: 点击生成后按钮显示加载动画，提示"正在生成视频，这可能需要几分钟..."（视频生成耗时较长，需给予明确预期）
  - **视频结果展示**: 生成成功后展示视频播放器（原生 `<video>` 控件），支持播放/暂停/全屏
  - **视频下载**: 视频播放器旁提供下载按钮，点击触发浏览器下载（文件名：`agnes-video-{timestamp}.mp4`）
  - **生成历史**: 当前会话中生成的视频以缩略图列表形式展示在结果区下方，支持切换查看
  - **Token 未配置引导**: 若 Token 未配置，输入框禁用，显示提示"请先配置 API Token"
  - **错误处理**: 生成失败时 toast 提示具体错误信息，输入框保持可用状态
